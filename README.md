# mmm 언어 & mmm_L IDE

> **대한민국이 낳은 위대한 난해 프로그래밍 언어, mmm**
> 엄랭에서 영감을 받은 한국어 에소테릭 언어입니다.

---

## mmm 언어 특징

- 모든 문장은 반드시 `~`로 끝나야 합니다.
- `print(값)~` 으로 출력합니다.
- 문자열은 `'작은따옴표'` 또는 `"큰따옴표"` 사용.
- `print(1 + 1)~` → `2` 출력.
- 변수 이름은 숫자로 시작할 수 없습니다.
- 나머지는 전부 어렵고 긴 한국어 키워드를 사용합니다.

---

## 키워드 목록

| 키워드 | 역할 |
|---|---|
| `변수야나와라 이름 = 값~` | 변수 선언 |
| `print(값)~` | 출력 |
| `이봐만약에(조건)~` | if 문 시작 |
| `아니면어쩔건데~` | else |
| `이봐끝났어~` | if 문 종료 |
| `반복해라(N)~` | N번 반복 (for) |
| `반복끝~` | for 반복 종료 |
| `계속해라(조건)~` | while 반복 |
| `그만해~` | while 반복 종료 |
| `참` | true |
| `거짓` | false |
| `그리고` | and |
| `아니면` | or |
| `부정` | not |

---

## 예제

```mmm
# 변수 선언
변수야나와라 이름 = '세계'~
print('안녕, ' + 이름)~

# 반복
반복해라(3)~
    print('반복!')~
반복끝~

# 조건문
변수야나와라 점수 = 85~
이봐만약에(점수 >= 80)~
    print('통과!')~
아니면어쩔건데~
    print('탈락!')~
이봐끝났어~

# while 반복
변수야나와라 i = 1~
계속해라(i <= 5)~
    print(i)~
    변수야나와라 i = i + 1~
그만해~
```

## mmm_L IDE 실행 방법

```bash
# 의존성 설치
npm install

# 개발 모드 실행
npm start

# 빌드 (현재 플랫폼)
npm run build

# Windows exe 빌드
npm run build:win

# macOS 빌드
npm run build:mac
```

> **오프라인 지원**: 빌드된 앱은 인터넷 연결 없이도 동작합니다.
